W\&B Weave는 더 나은 언어 모델 앱을 구축하는 데 도움을 줍니다. Weave를 사용하여 앱을 추적, 테스트 및 개선하세요:

* **추적 및 관찰**: 실시간 시스템에서 언어 모델 호출이 어떻게 작동하는지 확인하세요.
* **변경 사항 테스트**: 새로운 프롬프트, 데이터 및 모델을 안전하게 시도해 보세요.
* **테스트 실행**: Playground에서 모델과 프롬프트를 테스트하세요.
* **성능 확인**: 도구를 사용하여 앱의 성능을 확인하세요.
* **안전성 추가**: 콘텐츠 필터와 프롬프트 가드로 앱을 보호하세요.

다음을 통해 Weave를 코드에 연결하세요:

* [Python SDK](./reference/python-sdk/weave/index)
* [TypeScript SDK](./reference/typescript-sdk/weave/README)
* [Service API](./reference/service-api/index)

Weave는 다양한 언어 모델 제공업체, 로컬 모델 및 도구와 함께 작동합니다.

## 시작하기

Weave를 처음 사용하시나요? [Python 빠른 시작](./quickstart) 또는 TypeScript 빠른 시작으로 시작하세요.

## 고급 가이드

고급 주제 탐색:

* **통합**: Weave를 인기 있는 언어 모델 제공업체와 연결하세요.
* **[쿡북](/ko/cookbooks/Intro_to_Weave_Hello_Trace)**: Python과 TypeScript를 사용하여 Weave로 구축하세요. 대화형 노트북을 사용해 보세요.
* **W\&B AI Academy**: 고급 검색 시스템을 구축하고, 언어 모델 프롬프팅을 개선하며, 모델을 미세 조정하세요.

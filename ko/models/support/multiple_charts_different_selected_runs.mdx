W&B Reports를 사용하여 다음 단계를 따르세요.

1. 여러 패널 그리드를 만듭니다.
2. 필터를 적용하여 각 패널 그리드에 대해 원하는 run 세트를 선택합니다.
3. 패널 그리드 내에서 원하는 차트를 생성합니다.

트레이닝 도중 머신 연결이 끊겼을 가능성이 있습니다. [`wandb sync [PATH_TO_RUN]`](/ko/ref/cli/wandb-sync) 을 실행하여 데이터를 복구하세요. 해당 run의 경로는 실행 중인 Run ID와 일치하는 `wandb` 디렉토리의 폴더입니다.

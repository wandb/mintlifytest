W&B의 주요 엔지니어와 지원 스태프는 디버깅 목적으로 사용자 권한 하에 기록된 값에 엑세스합니다. 모든 데이터 저장소는 저장 시 데이터를 암호화하고, 감사 로그는 엑세스를 기록합니다. W&B 직원의 완벽한 데이터 보안을 위해 자체 호스팅 솔루션을 라이선스하여 자체 인프라 내에서 W&B 서버를 실행하십시오.

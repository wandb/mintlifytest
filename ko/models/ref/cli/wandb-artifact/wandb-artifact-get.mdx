**사용법**

`wandb artifact get [OPTIONS] PATH`

**요약**

wandb에서 아티팩트 다운로드


**옵션**

| **옵션** | **설명** |
| :--- | :--- |
| `--root` | 아티팩트를 다운로드할 디렉토리 |
| `--type` | 다운로드할 아티팩트의 유형 |

**사용법**

`wandb pull [OPTIONS] RUN`

**요약**

Weights & Biases에서 파일 가져오기

**옵션**

| **옵션** | **설명** |
| :--- | :--- |
| `-p, --project` | 다운로드하려는 project입니다. |
| `-e, --entity` | 목록의 범위를 지정할 entity입니다. |

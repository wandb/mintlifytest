**사용법**

`wandb job [OPTIONS] COMMAND [ARGS]...`

**요약**

W&B job을 관리하고 보기 위한 코맨드

**옵션**

| **옵션** | **설명** |
| :--- | :--- |

**코맨드**

| **코맨드** | **설명** |
| :--- | :--- |
| create | wandb run 없이 소스에서 job을 생성합니다. |
| describe | Launch job을 설명합니다. |
| list | 프로젝트의 job을 나열합니다. |

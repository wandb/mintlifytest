graphs를 위한 Wandb 클래스입니다.

```python
Graph(
    format="keras"
)
```

이 클래스는 일반적으로 신경망 모델을 저장하고 표시하는 데 사용됩니다. 노드와 엣지의 배열로 그래프를 나타냅니다. 노드는 wandb에서 시각화할 수 있는 레이블을 가질 수 있습니다.

#### 예시:

keras 모델 가져오기:

```
Graph.from_keras(keras_model)
```

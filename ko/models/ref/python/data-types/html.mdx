임의의 HTML을 위한 Wandb 클래스입니다.

```python
Html(
    data: Union[str, 'TextIO'],
    inject: bool = (True)
) -> None
```

| ARG |  |
| :--- | :--- |
|  `data` |  (문자열 또는 IO 오브젝트) wandb에 표시할 HTML |
|  `inject` |  (부울) HTML 오브젝트에 스타일시트를 추가합니다. False로 설정하면 HTML이 변경되지 않고 전달됩니다. |

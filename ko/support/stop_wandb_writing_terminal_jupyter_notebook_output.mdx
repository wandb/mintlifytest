환경 변수 [`WANDB_SILENT`](/ko/guides/models/track/environment-variables) 를 `true` 로 설정하세요.

<Tabs>
<Tab title="Python">
```python
os.environ["WANDB_SILENT"] = "true"
```
</Tab>
<Tab title="Tab">
```python
%env WANDB_SILENT=true
```
</Tab>
<Tab title="Tab">
```shell
WANDB_SILENT=true
```
</Tab>
</Tabs>

개인 및 서비스 계정 키는 로테이션하거나 해지할 수 있습니다. 새 API 키 또는 서비스 계정 사용자를 생성한 다음 새 키를 사용하도록 스크립트 를 재구성합니다. 재구성 후 프로필 또는 팀에서 이전 API 키를 제거하세요.

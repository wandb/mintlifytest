다음 예제는 [`wandb.log()`](/ko/ref/python/log/)를 사용하여 여러 가지 방법으로 손실을 기록하는 방법을 보여줍니다.

<Tabs>
<Tab title="Tab">
```python
wandb.log({f"losses/loss-{ii}": loss for ii, 
  loss in enumerate(losses)})
```
</Tab>
<Tab title="Tab">
```python
# 손실을 히스토그램으로 변환합니다.
wandb.log({"losses": wandb.Histogram(losses)})  
```
</Tab>
</Tabs>

자세한 내용은 [로깅에 대한 문서](/ko/guides/models/track/log/)를 참조하세요.

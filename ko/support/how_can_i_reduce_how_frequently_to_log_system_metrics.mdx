[시스템 메트릭](/ref/system-metrics)을 로깅하는 빈도를 설정하려면 `_stats_sampling_interval`을 초 단위로 나타낸 부동 소수점 숫자로 설정하세요. 기본값: `10.0`.

```python
wandb.init(settings=wandb.Settings(x_stats_sampling_interval=30.0))
```

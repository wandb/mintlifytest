`wandb.plot.line_series()`를 사용하여 여러 줄로 된 사용자 정의 차트를 만드세요. 라인 차트를 보려면 [프로젝트 페이지](/ko/guides/models/track/project-page)로 이동하세요. 범례를 추가하려면 `wandb.plot.line_series()`에 `keys` 인수를 포함하세요. 예:

```python
wandb.log(
    {
        "my_plot": wandb.plot.line_series(
            xs=x_data, ys=y_data, keys=["metric_A", "metric_B"]
        )
    }
)
```

**여러 줄** 탭 아래의 [여기](/ko/guides/models/track/log/plots#basic-charts)에서 여러 줄 플롯에 대한 추가 세부 정보를 참조하세요.

다음 방법 중 하나를 사용하여 로그인 URL을 설정합니다.

- [환경 변수](/ko/guides/models/track/environment-variables) `WANDB_BASE_URL`을 Server URL로 설정합니다.
- [`wandb login`](/ko/ref/cli/wandb-login)의 `--host` 플래그를 Server URL로 설정합니다.

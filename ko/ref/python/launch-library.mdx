## 클래스

[`class LaunchAgent`](./launchagent): run queue에 주어진 run을 폴링하고 wandb Launch에 대한 run을 시작하는 Launch 에이전트 클래스입니다.

## 함수

[`launch(...)`](./launch): W&B Launch 실험을 시작합니다.

[`launch_add(...)`](./launch_add): 소스 URI, job 또는 docker_image를 사용하여 W&B Launch 실험을 대기열에 추가합니다.

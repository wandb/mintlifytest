**사용법**

`wandb server [OPTIONS] COMMAND [ARGS]...`

**요약**

로컬 W&B 서버를 운영하기 위한 코맨드

**옵션**

| **옵션** | **설명** |
| :--- | :--- |

**코맨드**

| **코맨드** | **설명** |
| :--- | :--- |
| start | 로컬 W&B 서버 시작 |
| stop | 로컬 W&B 서버 중지 |

**사용법**

`wandb init [OPTIONS]`

**요약**

Weights & Biases 와 함께 디렉토리를 구성합니다.

**옵션**

| **옵션** | **설명** |
| :--- | :--- |
| `-p, --project` | 사용할 프로젝트입니다. |
| `-e, --entity` | 프로젝트 범위를 지정할 엔티티입니다. |
| `--reset` | 설정을 재설정합니다. |
| `-m, --mode` | "online", "offline" 또는 "disabled"가 될 수 있습니다. 기본값은 온라인입니다. |

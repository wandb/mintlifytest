**사용법**

`wandb server start [OPTIONS]`

**요약**

로컬 W&B 서버 시작


**옵션**

| **옵션** | **설명** |
| :--- | :--- |
| `-p, --port` | W&B 서버를 바인딩할 호스트 포트 |
| `-e, --env` | wandb/local에 전달할 환경 변수 |
| `--daemon / --no-daemon` | 데몬 모드로 실행 여부 |

---
title: wandb server
---

**Usage**

`wandb server [OPTIONS] COMMAND [ARGS]...`

**Summary**

Commands for operating a local W&B server


**Options**

| **Option** | **Description** |
| :--- | :--- |


**Commands**

| **Command** | **Description** |
| :--- | :--- |
| start | Start a local W&B server |
| stop | Stop a local W&B server |

## ようこそ

APIドキュメントを構築する方法は2つあります: [OpenAPI](https://mintlify.com/docs/api-playground/openapi/setup) と [MDXコンポーネント](https://mintlify.com/docs/api-playground/mdx/configuration)。スターターキットでは、以下のOpenAPI仕様を使用しています。

<Card title="Plant Storeエンドポイント" icon="leaf" href="https://github.com/mintlify/starter/blob/main/api-reference/openapi.json">
  OpenAPI仕様ファイルを表示
</Card>

## 認証

すべてのAPIエンドポイントは、Bearerトークンを使用して認証され、仕様ファイルから取得されます。

```json
"security": [
  {
    "bearerAuth": []
  }
]
```

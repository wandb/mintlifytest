## 画像

### Markdownの使用

[markdown構文](https://www.markdownguide.org/basic-syntax/#images) を使用すると、次のコードで画像を追加できます

```md
![title](../images/logo-large.png)
```

画像ファイルのサイズは5MB未満である必要があることに注意してください。それ以外の場合は、[Cloudinary](https://cloudinary.com/) や [S3](https://aws.amazon.com/s3/) などのサービスでホスティングすることをお勧めします。そのURLを使用して埋め込むことができます。

### 埋め込みの使用

画像をより柔軟にカスタマイズするには、HTML埋め込みを使用して画像を追加することもできます

```html
<img height="200" src="./images/logo-large.png" />
```

### iFrame

ドキュメント内に別のHTMLページを読み込みます。主にビデオの埋め込みに使用されます。

```html
<iframe src="https://www.youtube.com/embed/4KzFe50RQkQ"> </iframe>
```

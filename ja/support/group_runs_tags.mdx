run は複数のタグを持つことができるため、タグによるグループ化はサポートされていません。これらの run に対して [`config`](/ja/guides/models/track/config) オブジェクトに値を追加し、この config 値でグループ化してください。これは [API](/ja/guides/models/track/config#set-the-configuration-after-your-run-has-finished) を使用して達成できます。

トレーニング中にマシンへの接続が失われた可能性があります。データを回復するには、[`wandb sync [PATH_TO_RUN]`](/ja/ref/cli/wandb-sync) を実行してください。 run へのパスは、進行中の Run ID に一致する `wandb` ディレクトリー内のフォルダーです。

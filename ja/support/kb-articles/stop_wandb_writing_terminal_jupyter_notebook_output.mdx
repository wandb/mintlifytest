環境変数 [`WANDB_SILENT`](/ja/guides/models/track/environment-variables) を `true` に設定します。

<Tabs>
<Tab title="Python">
```python
os.environ["WANDB_SILENT"] = "true"
```
</Tab>
<Tab title="Tab">
```python
%env WANDB_SILENT=true
```
</Tab>
<Tab title="Tab">
```shell
WANDB_SILENT=true
```
</Tab>
</Tabs>

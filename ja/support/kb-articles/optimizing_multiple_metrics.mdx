複数のメトリクスを単一の run で最適化するには、個々のメトリクスの加重平均を使用します。

```python
metric_combined = 0.3 * metric_a + 0.2 * metric_b + ... + 1.5 * metric_n
wandb.log({"metric_combined": metric_combined})
```

新しい組み合わせメトリクスをログし、それを最適化の目標として設定します:

```yaml
metric:
  name: metric_combined
  goal: minimize
```

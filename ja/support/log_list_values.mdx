これらの例は、異なる方法で損失をログとして記録する [`wandb.log()`](/ja/ref/python/log/) の使い方を示しています。
<Tabs>
<Tab title="Tab">
```python
wandb.log({f"losses/loss-{ii}": loss for ii, 
  loss in enumerate(losses)})
```
</Tab>
<Tab title="Tab">
```python
# 損失をヒストグラムに変換します
wandb.log({"losses": wandb.Histogram(losses)})  
```
</Tab>
</Tabs>

詳細は [ログのドキュメント](/ja/guides/models/track/log/) を参照してください。

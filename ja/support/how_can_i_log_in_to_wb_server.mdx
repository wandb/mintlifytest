ログイン URL は、次のいずれかの方法で設定します:

- [環境変数](/ja/guides/models/track/environment-variables) `WANDB_BASE_URL` をサーバーの URL に設定します。
- [`wandb login`](/ja/ref/cli/wandb-login) の `--host` フラグをサーバーの URL に設定します。

アーティファクトの `description`、`metadata`、および `alias` に希望する値を渡します。W&B サーバー上でアーティファクトを更新するには、`save()` メソッドを呼び出してください。W&B Run の間または Run の外でアーティファクトを更新することができます。

W&B Public API ([`wandb.Api`](/ja/ref/python/public-api/api)) を使用して、Run の外でアーティファクトを更新します。Artifact API ([`wandb.Artifact`](/ja/ref/python/artifact)) を使用して、Run の間にアーティファクトを更新します。

<Warning>
Model Registry にリンクされたアーティファクトのエイリアスを更新することはできません。
</Warning>

<Tabs>
<Tab title="During a run">
次のコード例は、[`wandb.Artifact`](/ja/ref/python/artifact) API を使用してアーティファクトの説明を更新する方法を示しています。

```python
import wandb

run = wandb.init(project="<example>")
artifact = run.use_artifact("<artifact-name>:<alias>")
artifact.description = "<description>"
artifact.save()
```
</Tab>
<Tab title="Outside of a run">
次のコード例は、`wandb.Api` API を使用してアーティファクトの説明を更新する方法を示しています。

```python
import wandb

api = wandb.Api()

artifact = api.artifact("entity/project/artifact:alias")

# 説明を更新する
artifact.description = "My new description"

# メタデータキーを選択的に更新する
artifact.metadata["oldKey"] = "new value"

# メタデータを完全に置き換える
artifact.metadata = {"newKey": "new value"}

# エイリアスを追加する
artifact.aliases.append("best")

# エイリアスを削除する
artifact.aliases.remove("latest")

# エイリアスを完全に置き換える
artifact.aliases = ["replaced"]

# すべてのアーティファクトの変更を保存する
artifact.save()
```

詳細は、Weights and Biases [Artifact API](/ja/ref/python/artifact) を参照してください。
</Tab>
<Tab title="With collections">
コレクションも単一のアーティファクトと同様に更新することができます。

```python
import wandb
run = wandb.init(project="<example>")
api = wandb.Api()
artifact = api.artifact_collection(type="<type-name>", collection="<collection-name>")
artifact.name = "<new-collection-name>"
artifact.description = "<This is where you'd describe the purpose of your collection.>"
artifact.save()
```
詳細は [Artifacts Collection](/ja/ref/python/public-api/api) リファレンスを参照してください。
</Tab>
</Tabs>

W&B Core は [W&B Models](/ja/guides/models/) と [W&B Weave](/ja/guides/weave/) をサポートする基盤フレームワークであり、[W&B Platform](/ja/guides/hosting/) によってサポートされています。

<Frame>
    <img src="/images/general/core.png"  />
</Frame>

W&B Core は、ML ライフサイクル全体にわたる機能を提供します。W&B Core を使用すると、次のことができます:

- 完全なリネージトレースを使用して [ML パイプラインのバージョン管理と管理](/ja/guides/core/artifacts/) を行い、簡単に監査と再現性を確保します。
- [インタラクティブで設定可能な可視化](/ja/guides/models/tables/)を使用して、データとメトリクスを探索および評価します。
- [レポートを生成し、組織全体で洞察を文書化し共有します](/ja/guides/core/reports/)。非技術系の利害関係者にも理解しやすい統計化された形式でライブレポートを生成することで達成します。
- [カスタムニーズに合わせたデータのクエリと可視化を作成します](/ja/guides/models/app/features/panels/query-panels/)。
- [秘密情報をシークレットで保護します](/ja/guides/core/secrets)。
- [モデル CI/CD](/ja/guides/core/automations/) のためのキーとなるワークフローをトリガーするオートメーションを設定します。

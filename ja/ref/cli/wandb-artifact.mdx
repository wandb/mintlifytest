**使用方法**

`wandb artifact [OPTIONS] COMMAND [ARGS]...`

**概要**

アーティファクトと対話するためのコマンド

**オプション**

| **オプション** | **説明** |
| :--- | :--- |

**コマンド**

| **コマンド** | **説明** |
| :--- | :--- |
| cache | アーティファクトキャッシュと対話するためのコマンド |
| get | wandb からアーティファクトをダウンロード |
| ls | wandb プロジェクト内のすべてのアーティファクトをリスト表示 |
| put | wandb にアーティファクトをアップロード |

**使用法**

`wandb init [OPTIONS]`

**概要**

Weights & Biases でディレクトリーを設定

**オプション**

| **オプション** | **説明** |
| :--- | :--- |
| `-p, --project` | 使用するプロジェクト。 |
| `-e, --entity` | プロジェクトを範囲指定するためのエンティティ。 |
| `--reset` | 設定をリセット |
| `-m, --mode` | "online"、"offline" または "disabled" を指定可能。デフォルトは online。 |

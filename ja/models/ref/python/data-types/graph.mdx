グラフ用の Wandb クラス。

```python
Graph(
    format="keras"
)
```

このクラスは通常、ニューラルネットモデルを保存し表示するために使用されます。ツリーをノードとエッジの配列として表現します。ノードには、wandb で可視化できるラベルを持たせることができます。

#### 例:

Keras モデルをインポート:

```
Graph.from_keras(keras_model)
```

スイープに関連付けられた一連の runs。

```python
Sweep(
    client, entity, project, sweep_id, attrs=None
)
```

#### 例:

次のようにインスタンス化します:

```
api = wandb.Api()
sweep = api.sweep(path / to / sweep)
```

| 属性 |  |
| :--- | :--- |
|  `runs` |  (`Runs`) run のリスト |
|  `id` |  (str) スイープの id |
|  `project` |  (str) プロジェクトの名前 |
|  `config` |  (str) スイープ設定の辞書 |
|  `state` |  (str) スイープの状態 |
|  `expected_run_count` |  (int) スイープの予想される run の数 |

**使用方法**

`wandb artifact get [OPTIONS] PATH`

**概要**

wandb からアーティファクトをダウンロードします

**オプション**

| **オプション** | **説明** |
| :--- | :--- |
| `--root` | アーティファクトをダウンロードしたいディレクトリー |
| `--type` | ダウンロードするアーティファクトのタイプ |

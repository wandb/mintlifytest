---
title: How do I stop wandb from writing to my terminal or my Jupyter notebook output?
---

Set the environment variable [`WANDB_SILENT`](/guides/models/track/environment-variables.md) to `true`.

<Tabs>
  <Tab title="Python">
```python
os.environ["WANDB_SILENT"] = "true"
```
  </Tab>
  <Tab title="Notebook">
```python
%env WANDB_SILENT=true
```
  </Tab>
  <Tab title="Command-Line">
```shell
WANDB_SILENT=true
```
  </Tab>
</Tabs>

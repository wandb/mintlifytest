---
title: wandb init
---

**Usage**

`wandb init [OPTIONS]`

**Summary**

Configure a directory with Weights & Biases


**Options**

| **Option** | **Description** |
| :--- | :--- |
| `-p, --project` | The project to use. |
| `-e, --entity` | The entity to scope the project to. |
| `--reset` | Reset settings |
| `-m, --mode` | Can be "online", "offline" or "disabled". Defaults to   online. |

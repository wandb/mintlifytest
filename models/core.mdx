W&B Core is the foundational framework supporting [W&B Models](/guides/models/) and [W&B Weave](/guides/weave/), and is itself supported by the [W&B Platform](/guides/hosting/). 

<Frame>
    <img src="/images/general/core.png" alt="W&B Core framework diagram"  />
</Frame>

W&B Core provides capabilities across the entire ML lifecycle. With W&B Core, you can:

- [Version and manage ML](/guides/artifacts/) pipelines with full lineage tracing for easy auditing and reproducibility.
- Explore and evaluate data and metrics using [interactive, configurable visualizations](/guides/tables/).
- [Document and share](/guides/reports/) insights across the entire organization by generating live reports in digestible, visual formats that are easily understood by non-technical stakeholders.
- [Query and create visualizations of your data](/guides/app/features/panels/query-panels/) that serve your custom needs.
- [Protect sensitive strings using secrets](/guides/secrets/).
- Configure automations that trigger key workflows for [model CI/CD](/guides/automations/).
